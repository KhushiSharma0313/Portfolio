## Description

This project consists of a static website built using HTML, CSS, and JavaScript. It serves as a professional portfolio for Khushi Sharma, providing information about her background, skills, experiences, and projects. The website is designed to be responsive, ensuring a seamless experience across various devices.

## Features

- **Navigation**: The website includes navigation menus both for desktop and mobile devices, ensuring easy access to different sections such as About, Experience, Projects, and Contact.
- **Profile Section**: It features a profile section with a brief introduction along with profile picture and links to download my CV and access my social media profiles.
- **About Section**: This section provides detailed information about my background, education, and experiences, along with a brief description of my skills and interests.
- **Experience Section**: My professional experiences are showcased in this section, including roles as a research assistant, data analyst intern, and software engineer intern.
- **Projects Section**: It highlights my recent projects, along with links to GitHub repositories or other relevant resources for further exploration.
- **Contact Section**: The website includes a contact section with links to my email and LinkedIn profile, enabling visitors to reach out easily.
